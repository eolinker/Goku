# Checker

##### 规则：

```
<p>=<pr>str<su>
```

##### 可能值：

p：^    !    ~    ~*

pr:  *    **    !    $

su: *

#### 匹配优先级（由上往下优先级递减）

```
1.全等匹配：=str(=可省略)
2.前缀匹配：^=str、      =str*(=可省略)
3.后缀匹配：^=*str、     =*str(=可省略)
4.子串匹配：=*str*(=可省略)
5.非等匹配：!=str
6.空值匹配：=$(=可省略)
7.存在匹配：=**(=可省略)
8.不存在匹配：=!(=可省略)
9.区分大小写的正则匹配：~=str
10.不区分大小写的正则匹配：~*=str
11.任意匹配：=(=可省略)、    =*(=可省略)
```
